146

147

     * \]

     * \[

     * \]

     * \[

     * \]

     * \[

     * \]

     * \]

     *

     * compute them directly, knowing that \(A = a^2 \omega^2\) and that

     * \(B = -\omega^2\). The complete algorithm is therefore:</p>
202 
     *

204 
     * \[ f(t_i) \]

     * \[ f'(t_i) = \frac{f (t_{i+1}) - f(t_{i-1})}{t_{i+1} - t_{i-1}} \]

     * \[ x_i = t_i  - t_1 \]

     * \[ z_i = \int_{t_1}^{t_i} f'^2(t) dt \]
